Грамматика G[E]:

E => TA
A => ε | + TA | - TA
T => OB
B => ε | *OB | / OB
O => num | id | (E)
num - числовая константа Ц{Ц}
id - идентификатор Б{Б|Ц}
Б - [A, B, …, Z], Ц - [0, 1, …, 9]

Язык

L(G[Е]) = {x ∈ VT* | E =>* x}
VT = {num, id, +, -, *, /, (, )}
VN = {E, A, T, B, O}
Е - начальный нетерминал

Тестовый пример

Выражение: (45+a)*25

Последовательность разбора:
E__T__O__(__E__T__O__num__B__ε__A__+__T__O__id__B__ε__A__ε__)__B__*__O__num__B__ε__A__ε

Классификация

Согласно классификации Хомского, грамматика G[E] является контекстно-свободной.
Все правила относятся к классу продукций: A => a, A ∈ Vn, а ∈ V*
